Perguntas Frequentes

Encontre respostas para as dúvidas mais comuns sobre nossos cursos e processo de habilitação

Processo de Inscrição +

Qual é a idade mínima para começar as aulas? +

Para categoria B (carros), você pode se inscrever aos 17 anos e meio, mas só poderá realizar o exame prático aos 18 anos. Para motos (A1), a idade mínima é 16 anos.

Número de Aulas por Curso +

Quantas aulas são necessárias para a categoria A1 (motos)? +

Para a categoria A1, são necessárias 28 aulas teóricas e 12 aulas práticas, totalizando 40 aulas.

Prazo para Conclusão do Curso +

Qual é o prazo máximo para concluir o processo de habilitação? +

O prazo máximo para concluir todo o processo de habilitação, desde a inscrição até a aprovação no exame prático, é de 2 anos.
